Class: ColorTable

ColorTable

new ColorTable()

Creates a new ColorTable. Use a color table to create a map of keywords to color ranges. Instead of manually passing start and end colors when creating color palettes, you can use getColor() and pass a keyword to receive the start and end colors.

Example
var heat = defaultColors.getColor('heat');
console.log(heat.startColor); // -> [255, 132, 86]
console.log(heat.endColor); // -> [175, 47, 0]

getColor(options, startColor, endColor) → {Object|Array}

Returns start and end colors from a key in the color table.
Parameters:
Name Type Description
options Object A set of options. Required: options.name {number} The name of the entry in the color table. Optional: options.startColor {boolean} Pass true to only return the start color. options.endColor {boolean} Pass true to only return the end color.
startColor Array An array representing the start color. ex: [255, 100, 50].
endColor Array An array representing the end color. ex: [155, 50, 10].

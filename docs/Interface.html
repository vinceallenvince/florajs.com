<static> checkRequiredParams(optionsPassed, optionsRequired, opt_from) → {boolean}

Compares passed options to a set of required options.
Parameters:
Name Type Argument Description
optionsPassed Object An object containing options passed to a function.
optionsRequired Object An object containing a function's required options.
opt_from string <optional>
 A message or identifier from the calling function.
Source:
  • interface.js, line 5
Returns:
Returns true if all required options are present and of the right data type. Returns false if any required options are missing or are the wrong data type. Also returns false if any of the passed options are empty.
Type
boolean
Example
optionsPassed = {
 form: document.getElementById("form"),
   color: "blue",
 total: "a whole bunch",
  names: ["jimmy", "joey"]
}
optionsRequired = {
  form: "object",
  color: "string",
  total: "number|string",
  names: "array"
}
checkRequiredOptions(optionsPassed, optionsRequired) returns true in this case.

Notice you can separate required options data types with a pipe character to allow multiple data types.
